# Steam Games Analysis

Đề tài phân tích dữ liệu Steam Games nhằm tìm hiểu mỗi liên hệ giữa độ phổ biến của game (estimated owners) với giá bán, đánh giá người dùng, thể loại, độ dài, độ chi tiết của mô tả và các yếu tố kỹ thuật khác.

## Dữ liệu

- Tệp dữ liệu: `games.csv` đặt cùng cấp với chương trình chính.
- Nguồn dữ liệu: https://www.kaggle.com/datasets/fronkongames/steam-games-dataset.
- Chương trình chính: `assignment.py`
- Biểu đồ: `charts/`
- Slide báo cáo: https://canva.link/5am5kuarrhzzri9

## Cài đặt

```bash
pip install -r requirement.txt
```

## Cách chạy

```bash
python assignment.py
```
